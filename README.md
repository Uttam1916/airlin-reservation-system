# Airline Reservation System (C Mini Project)

## Structure
- **Graph:** Stores flight routes.
- **Queue:** Manages waiting list.
- **Hash Table:** Manages passenger info.
- **Reservation Module:** Handles booking/cancellation.

## Build & Run
```bash
make
./airline
```

## Files
- Headers in `include/`
- Implementations in `src/`
- Entry point: `main.c`
